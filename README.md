Columbia, MO Viral Wastewater Surveillance (2024–2025)

This repository contains input data and analysis code used to generate the figures for the associated manuscript.

The repository is intended to support full reproducibility of all figures included in the manuscript.

---

# Contents

- Input data files used in the analyses
- An R Markdown script that generates all manuscript figures
- A `ManuscriptFigures/` directory where all figures are written

---

# Reproducing the Figures

# 1. Download the repository

Download the full repository as a ZIP file from GitHub, or clone it using git:

```bash
git clone https://github.com/ClaytonRushford/Columbia_MO_viral_surveillance_2024-2025.git

After downloading, ensure that the repository has the following structure:

Columbia_MO_viral_surveillance_2024-2025/
├── Manuscript_2026-01-22.Rmd
├── ManuscriptFigures/
├── (input data files and folders)

The ManuscriptFigures/ directory must exist in the same directory as the R Markdown file.
All figures generated by the script will be written to this folder.
Open Manuscript_2026-01-22.Rmd in RStudio and knit the document, or run it directly from R.
Running the script will regenerate all figures included in the manuscript, which will be saved to the ManuscriptFigures/ directory.
